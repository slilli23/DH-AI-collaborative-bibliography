# DH-AI Collaborative Bibliography

The Italian Observatory on AI and Digital Humanities (DH-AI) is a project launched
in late 2025 as a Special Interest Group (SIG) of the Italian Association for Digital
Humanities and Digital Culture (AIUCD). Among the SIG's core activities is the creation
of an open collaborative bibliography on AI and DH topics.

This repository contains supplementary material for the paper presented at the
EADH 2026 Conference.

## Contents

1. `bibliography_export_2026-06-30.csv` — DH-AI Zotero bibliography (249 items).
2. `tagging_guidelines.pdf` — instructions issued to contributors for the
   collaborative tagging task (language: italian).
3. `item_types.html` — distribution of items by type (interactive chart).
4. `language_attribution.csv` — language assigned to each item, joinable to the
   export on `Key`. Taken from the Zotero `Language` field where compiled (173
   items) and inferred from title and abstract with `langdetect` where missing
   (76 items); the `source` column records which.
5. `tag_mapping.csv` — tag normalisation table. The draft was generated by an LLM
   (Claude Opus 4.8) and subsequently validated by hand. Each of the 391 original
   tags is assigned an action (keep / merge / facet / drop) and, where applicable,
   a canonical concept.
6. `network_edgelist.csv`, `network_metrics.csv`, `network.html` — co-occurrence
   network of the normalised concepts.

## Reproducibility

Scripts: `analysis.py`, `normalize_tags.py`, `network_analysis.py`.
Concept co-occurrence is projected from the bipartite item–concept graph with Newman
weighting (an item indexing *n* concepts contributes 1/(*n*−1) to each pair). Concepts
occurring in fewer than 2 items are excluded. Communities are detected with the Louvain
algorithm (seed 42). Python 3.12; pandas, plotly, langdetect, networkx, pyvis.
